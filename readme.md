# ➕ Simple Calculator

A command-line calculator built in Python that performs basic arithmetic operations using `match-case` statements.

## Features

- Addition, subtraction, multiplication, and division
- Handles invalid number input (`ValueError`)
- Handles division by zero (`ZeroDivisionError`)
- Handles invalid operation choice

```bash
python calculator.py
```
